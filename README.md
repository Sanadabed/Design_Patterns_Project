# LaptopStore – Budget-Based Laptop Recommendation System

## Project Idea
User enters budget → System recommends suitable laptop (Budget, Study, Business, Gaming)

## Pattern Used
Abstract Factory Pattern - Creates compatible product families (CPU, RAM, Storage, GPU, Screen)
